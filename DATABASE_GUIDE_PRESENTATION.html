PocketFlow Database Guide

Clear HTML presentation based on DATABASE_GUIDE.md

Overview

Engine: Dexie.js + IndexedDB | DB Name: pocketflow-budget-db | Scope: local storage

Profile Rules & Cutoffs Template & Builder Tracker Entries Unexpected Others

1) userProfiles

  • id (PK)
  • displayName
  • photoUrl (optional)
  • createdAt, updatedAt
  • Index: updatedAt

2) rules

  • id (PK), name, percent, itemCount
  • Rule names: Expenses, Savings, Wants
  • Validation: total percent must be 100

3) cycleCutoffs

  • id (PK), monthKey, slot, label, amount
  • Only 2 rows per monthKey (slot 1 and 2)
  • Default labels: 1st cutoff, 2nd cutoff

4) itemBuilders

  • id (PK), name, amount (optional)
  • categories[], isActive, hasChildItems
  • Reusable items + dropdown source when adding entries

5) itemBuilderChildren

  • id (PK), parentItemId (ref itemBuilders.id)
  • name, amount, createdAt
  • Used for child items under a parent

6) budgetEntries

  • id (PK), monthKey, ruleName, cutoff
  • name, amount, createdAt
  • Validation: per-cutoff total must not exceed cutoff amount

7) unexpectedExpenses

  • id (PK), monthKey, expenseName, amount
  • amount is negative when overrun exists
  • Only created when budget is exceeded

8) othersBudgets

  • id (PK), monthKey, budgetAllocated, createdAt
  • Optional monthly bucket

9) othersExpenses

  • id (PK), monthKey, expenseName, amount
  • Expense rows under Others budget

Relationships

Parent Child Link
itemBuilders itemBuilderChildren itemBuilderChildren.parentItemId -> itemBuilders.id
rules budgetEntries budgetEntries.ruleName -> rules.name
cycleCutoffs budgetEntries monthKey + cutoff/label
othersBudgets othersExpenses othersExpenses.monthKey -> othersBudgets.monthKey

Validation Rules

  • Rule percentages total must equal 100.
  • Only 2 cutoffs per month/cycle.
  • Per cutoff allocation cannot exceed cutoff amount.
  • Unexpected expense is created only for overruns (negative amount).
  • Item builder parent progress uses parent amount versus child totals.
  • First app use requires profile setup (display name + profile picture).
